# WriteMate

一款用來快速生成可列印 A4 文章字帖的 Web App。採用 React + Vite + TypeScript 與 pdf-lib，目前聚焦文章模板（橫式 / 直式），格子精確 15mm，並支援示範文字嵌入。

## 目前功能
- 僅文章模板：17×12（204 格），橫排（左至右）或直排（右至左）。
- 格線樣式：田字／米字／九宮格，虛線導引，雙層外框，可調格線色。
- 字體：內建 5 款；可調字級與文字色。辰宇落雁體 Thin 會子集嵌入 PDF。
- 參考層：可開關與調整透明度，直接重用輸入文字做示範。
- 標頭與留白：頂端姓名／日期標籤，額外 6mm 頂部留白；頁邊界 15mm；格子 15mm。
- 預覽與匯出：即時 SVG 預覽，60-140% 縮放；單頁 A4 PDF 匯出（pdf-lib + fontkit）。
- 狀態與預設：設定存於 LocalStorage，最多保留 3 組快速預設；文字超過 204 格時提示，匯出仍僅第一頁。

## 技術棧
- React 18、Vite、TypeScript
- Zustand：設定與預設管理
- pdf-lib + fontkit：向量 PDF 與字型嵌入
- SVG：螢幕預覽

## 快速開始
### 環境需求
- 建議 Node.js 18+ 

### 安裝
```bash
cd frontend
npm install
```

### 開發伺服器
```bash
cd frontend
npm run dev
```
依終端顯示網址開啟（預設 http://localhost:5173）。

### 建置
```bash
cd frontend
npm run build
```
產出於 `frontend/dist`。

## 使用提示
- 選擇橫式或直式模板，貼上文字後可調格線樣式、色彩與字型設定。
- 文字請控制在 204 字以內避免溢出；超過時僅匯出首頁並顯示警示。
- 列印時選擇「實際大小」或關閉縮放，才能保持 15mm 格子。

## 已知限制
- 目前僅單頁 PDF，尚未自動分頁。
- 不支援自訂字型上傳；僅內建字型，且僅辰宇落雁體會嵌入。
- 無行動 App 或後端服務，全部運行於前端。
